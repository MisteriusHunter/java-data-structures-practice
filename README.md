# 📊 Java Iteration & Array Practice

This repository contains practical examples of **Array management** and **Iteration techniques** in Java, focusing on clean and readable code using modern syntax.

## 🚀 Technical Concepts
- **Array Initialization**: Demonstrates both fixed-size array allocation and shorthand literal initialization.
- **Enhanced For-Loop (For-Each)**: Implementation of the most efficient way to iterate through collections without manual index management, reducing potential "Off-by-one" errors.
- **Data Handling**: Practice with String arrays to simulate real-world data lists.

## 🛠️ Included Examples
1. **Country List**: Manual array population and sequential printing.
2. **Animal Registry**: Comparison between traditional `for` loops and the more concise `for-each` approach.
